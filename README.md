# Astronaut Health and Mission Monitoring

**Description:**  
A basic web application to manage astronaut data. Users can add, view, and operate on astronaut details. The project also includes an AI-powered chat interface.

---

## Features
- Add, view, and manage astronaut data  
- AI chat interface  
- Organized folder structure for models, routes, and services  
- Built with Node.js
